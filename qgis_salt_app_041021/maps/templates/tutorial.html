TUTORIAL

Abridged Tutorial of how this app was created

  1.  I first started with the data file type as a tiff (raster) file, so I uploaded them to QGIS. I classified the data into 8 valued classes for the 8 different land-types found in the map.


  2.  I then used location methods such as "getLocation(), showPosition() and getMap()" to get the location of the user and then I outputted the location on the google maps.


  3.  I used PostGreSQL Server Database (open source) to store the data. It can be used to read big Tif files. In this database, I put the raster files (tif files) and then compressed it (in QGIS) so I can insert the raster data into a database. To insert the rastor file into the database, we used this command:

    We used raster2pgsql to upload the rastor file into postgre sql. The file name is kent_compressed, the database name is kent_2017 and the user name is 'U postgres'

    We have a localhost and the port number is 5432. Commands to insert raster file data into the postgre sql database:

    Go to the root folder   :  C:\Program Files\PostgreSQL\11\bin

    Insert data  :  raster2pgsql -I -C -M -t 50x50 "C:\Somerset_2017.tif" -F | psql -d Kent -U postgres -h localhost -p 5432

    I downloaded the django module to run the command prompt (I installed django using pip install in the command prompt) In the requirements file in my web app folder, it has all the libraries we need so I imported the libraries in command prompt. I am using Visual Studio and Visual Studio Code for the frontend and backend along with several libraries and functions.
